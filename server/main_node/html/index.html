DARLMID is a complete reinforcement learning scenario in-a-box, designed to rapidly allow researchers to process user demonstrations and utilize them towards training an autonomous driving agent within a 2D environment.

In contrast to other RL experiments and frameworks, DARLMID evaluates demonstration quality to filter and better utilize malicious and/or incompetent demonstrations in order to improve an agent's learning rate in a varietry of circumstances.

How it works


Many reinforcement learning (RL) projects utilize demonstrations provided by humans in order to increase the learning rate of an agent. However, the implicit assumption is that the expert giving the demonstration is providing a correct set of actions, even if these are suboptimal.

In contrast, DARLMID makes an assumption that the user is providing malicious or incompetent instructions at some or all points.

NOTE: DARLMID is still in the development stage. As such, some functionality may be missing. This program should not be utilized in its current condition without modification.
